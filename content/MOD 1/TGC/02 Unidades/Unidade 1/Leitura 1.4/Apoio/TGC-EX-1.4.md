### 1
> [!question] Exercício 1
> Qual o tipo de software que permite que um computador seja utilizado, ou seja, que é indispensável para o funcionamento de um computador?
>
> **A.**  
> Software primário.
>
> **B.**  
> Software aplicativo.
>
> **C.**  
> Software livre.
>
> **D.**  
> Software básico.
>
> **E.**  
> Software original.
>> [!question]- Resposta
>>
>> **Alternativa correta: D**
>>
>> **A. Errada.**  
>> “Software primário” não é a classificação usual para o software indispensável ao funcionamento do computador.
>>
>> **B. Errada.**  
>> Software aplicativo é o usado para tarefas específicas, como editores de texto, navegadores e planilhas, mas não é o que viabiliza o funcionamento básico do computador.
>>
>> **C. Errada.**  
>> Software livre se refere ao tipo de licença e liberdade de uso, modificação e distribuição, e não à função essencial de operar o computador.
>>
>> **D. Certa.**  
>> Software básico é o conjunto essencial para o funcionamento do computador, como o sistema operacional, que permite ao usuário utilizar a máquina.
>>
>> **E. Errada.**  
>> “Software original” não define a categoria de software indispensável ao funcionamento do computador.
>> 
>> Os softwares básicos são responsáveis por tornar o computador funcional, ou seja, realizar a integração entre hardware e software.

### 2
> [!question] Exercício 2
> Qual a definição adequada para software aplicativo?
>
> **A.**  
> É um tipo de software desenvolvido para realizar tarefas específicas para os usuários, auxiliando na realização de determinadas tarefas.
>
> **B.**  
> É um tipo de software que permite inicializar o computador.
>
> **C.**  
> É um tipo de software que permite a análise do desempenho do sistema operacional.
>
> **D.**  
> É um tipo de software multitarefa.
>
> **E.**  
> É um tipo de software desenvolvido para ser utilizado em equipamentos eletrônicos que não sejam computadores.
>> [!question]- Resposta
>>
>> **Alternativa correta: A**
>>
>> **A. Certa.**  
>> Software aplicativo é feito para ajudar o usuário a executar tarefas específicas, como editar textos, navegar na internet, fazer cálculos ou criar apresentações.
>>
>> **B. Errada.**  
>> Inicializar o computador é função mais ligada ao software básico, como firmware e sistema operacional, não à definição de software aplicativo.
>>
>> **C. Errada.**  
>> Isso descreve mais uma ferramenta utilitária ou de diagnóstico, não a definição geral de software aplicativo.
>>
>> **D. Errada.**  
>> “Multitarefa” é uma característica que pode existir em sistemas e programas, mas não define o que é software aplicativo.
>>
>> **E. Errada.**  
>> Software aplicativo não é definido por ser usado fora de computadores, mas por sua função de atender tarefas do usuário.
>> 
>> Os processadores de texto, as planilhas de cálculo e as bases de dados são software aplicativos, o que só vem confirmar que as aplicações computacionais possibilitam a automatização de tarefas.

### 3
> [!question] Exercício 3
> Como é chamado o tipo de aquisição de software em que o software é distribuído gratuitamente, mas para ter acesso à sua documentação e a funcionalidades adicionais é necessário pagar?
>
> **A.**  
> Freeware.
>
> **B.**  
> Shareware.
>
> **C.**  
> Houseware.
>
> **D.**  
> Firmware.
>
> **E.**  
> Software livre.
>> [!question]- Resposta
>>
>> **Alternativa correta: B**
>>
>> **A. Errada.**  
>> Freeware é um software distribuído gratuitamente, em geral para uso sem cobrança, mas isso não define o modelo de liberar uma versão limitada e cobrar por recursos extras.
>>
>> **B. Certa.**  
>> Shareware é o software distribuído gratuitamente de forma parcial ou para teste, mas que exige pagamento para acesso completo, documentação ou funcionalidades adicionais.
>>
>> **C. Errada.**  
>> “Houseware” não é uma classificação usual de software nesse contexto.
>>
>> **D. Errada.**  
>> Firmware é o software embutido em dispositivos eletrônicos, responsável por controlar seu funcionamento básico.
>>
>> **E. Errada.**  
>> Software livre se refere à liberdade de uso, estudo, modificação e distribuição, e não a esse modelo comercial de teste com pagamento posterior.
>> 
>> É uma categoria de software que muitas vezes é confundida com o freeware. Um software shareware é distribuído gratuitamente como o freeware, mas, gostando, o usuário tem que desembolsar uma quantia para adquirir a autorização e a documentação dele.

### 4
> [!question] Exercício 4
> Qual dos nomes de software abaixo é um nome de sistema operacional?
>
> **A.**  
> Microsoft Excel.
>
> **B.**  
> Mozila Firefox.
>
> **C.**  
> Corel Draw.
>
> **D.**  
> Linux.
>
> **E.**  
> Micorsoft Word.
>> [!question]- Resposta
>>
>> **Alternativa correta: D**
>>
>> **A. Errada.**  
>> Microsoft Excel é um aplicativo de planilhas, não um sistema operacional.
>>
>> **B. Errada.**  
>> Mozilla Firefox é um navegador de internet, não um sistema operacional.
>>
>> **C. Errada.**  
>> Corel Draw é um software de design gráfico, não um sistema operacional.
>>
>> **D. Certa.**  
>> Linux é o nome de um sistema operacional.
>>
>> **E. Errada.**  
>> Microsoft Word é um editor de texto, não um sistema operacional.
>> 
>> O Linux é um sistema operacional distribuído gratuitamente. Em termos de popularidade existem também os sistemas operacionais da plataforma Linux, que são baseados em software livre e têm diversas comunidades de desenvolvimento.

> [!question] Exercício 5
> Quais os nomes dos dois processadores de texto mais utilizados atualmente?
>
> **A.**  
> Excel e Calc.
>
> **B.**  
> Corel Draw e Photoshop.
>
> **C.**  
> Mozila Firefox e Google Chrome.
>
> **D.**  
> Linux e Windows.
>
> **E.**  
> Microsoft Word e Write (da BrOffice).
>> [!question]- Resposta
>>
>> **Alternativa correta: E**
>>
>> **A. Errada.**  
>> Excel e Calc são programas de planilha eletrônica, não processadores de texto.
>>
>> **B. Errada.**  
>> Corel Draw e Photoshop são softwares de edição gráfica e imagem, não processadores de texto.
>>
>> **C. Errada.**  
>> Mozilla Firefox e Google Chrome são navegadores de internet, não processadores de texto.
>>
>> **D. Errada.**  
>> Linux e Windows são sistemas operacionais, não processadores de texto.
>>
>> **E. Certa.**  
>> Microsoft Word e Write (da BrOffice/LibreOffice) são processadores de texto, usados para criar e editar documentos.
>> 
>> O mercado atualmente é dominado por dois aplicativos. São eles: o Word, da Microsoft e o Write, do BR-Office. Os processadores de texto são um tipo de software amplamente utilizado em computadores. Eles permitem criar, editar, armazenar e imprimir textos e gráficos em um documento
