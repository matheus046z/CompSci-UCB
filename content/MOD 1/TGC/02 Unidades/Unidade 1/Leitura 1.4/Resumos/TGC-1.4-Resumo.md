# Resumo — Software: conceitos, tipos e exemplos

## 1. O que é software

> [!info] Conceito
> **Software** é o conjunto de instruções lógicas que faz o computador executar tarefas. Ele não é a parte física da máquina, mas sim a parte “inteligente” que orienta o funcionamento do sistema.

O material apresenta o software como uma sequência de instruções usada para manipular, redirecionar ou modificar dados e informações. Em termos simples, é o que dá utilidade ao computador, porque permite que os componentes físicos sejam usados de forma organizada. Sem software, o hardware existe, mas não consegue atender às necessidades do usuário.

Além disso, o conteúdo destaca que o software está presente no cotidiano e não apenas nos computadores. Sempre que uma pessoa interage com um equipamento eletrônico, há algum tipo de software envolvido nessa interação.

> [!tip] Resumindo
> O **hardware** é a parte física; o **software** é a parte lógica que faz a máquina funcionar e executar tarefas.

---

## 2. Os dois grandes grupos: software básico e software aplicativo

> [!info] Conceito
> Os softwares são organizados em dois grupos principais: **software básico** e **software aplicativo**.

O software básico é o conjunto de programas que torna o computador utilizável. Ele gerencia o funcionamento do sistema e cria a ponte entre o usuário e a máquina. Já o software aplicativo é voltado para tarefas específicas do usuário, como escrever textos, fazer cálculos, criar apresentações ou acessar a internet.

Essa divisão é central no material porque ajuda a entender que nem todo software tem a mesma função. Alguns existem para fazer o computador operar; outros existem para ajudar o usuário a realizar atividades concretas.

> [!warning] Atenção
> Uma confusão comum é pensar que todo programa usado no computador tem a mesma função. O material diferencia claramente:
> - **software básico** = faz o computador funcionar;
> - **software aplicativo** = ajuda o usuário a executar tarefas.

> [!tip] Resumindo
> O software básico sustenta o uso do computador. O software aplicativo usa essa base para resolver necessidades práticas do usuário.

---

## 3. Software básico

> [!info] Conceito
> **Software básico** é o conjunto de programas responsáveis por integrar hardware e software, permitindo a operação do computador.

O conteúdo explica que o software básico torna o computador funcional. Ele é responsável por permitir a interação entre o usuário e os recursos da máquina. Entre seus exemplos, o mais importante é o **sistema operacional**, mas o material também inclui ambiente operacional, interface gráfica, recursos de rede e tradutores de linguagem.

A ideia central é que o usuário só consegue aproveitar os componentes físicos do computador porque existe um conjunto de programas que organiza, controla e apresenta esses recursos de forma compreensível.

> [!tip] Resumindo
> O software básico é a estrutura essencial que transforma um conjunto de peças eletrônicas em um sistema realmente utilizável.

---

## 4. Sistema operacional e suas funções

> [!info] Conceito
> O **sistema operacional** é um tipo de software básico que permite a operação do computador e a comunicação entre usuário e máquina.

Segundo o material, o sistema operacional executa tarefas fundamentais, como leitura e gravação de dados em disco, gerenciamento de impressão e execução de comandos. Também é ele que viabiliza a interação com o computador por meio de comandos e recursos que o usuário consegue acionar.

No desafio apresentado, o conteúdo destaca três tarefas básicas que um sistema operacional deve contemplar: oferecer uma interface para a comunicação com o usuário, gerenciar os dispositivos de hardware e gerenciar os sistemas de arquivos em disco. Em linguagem simples, isso significa permitir o uso do computador, controlar os equipamentos conectados e organizar arquivos e pastas.

> [!tip] Resumindo
> O sistema operacional é o principal mediador entre o usuário, o hardware e os arquivos do computador.

---

## 5. Tipos de sistemas operacionais

> [!info] Conceito
> O material classifica os sistemas operacionais conforme a forma de uso e execução das tarefas.

São apresentados três tipos principais. O sistema **monousuário** permite que apenas um usuário execute tarefas. O sistema **multitarefas** possibilita executar mais de uma tarefa ao mesmo tempo. Já o sistema **multiusuário** permite o acesso simultâneo de vários usuários ao computador.

Essas classificações ajudam a compreender que os sistemas operacionais podem ser diferentes não apenas pelo nome ou fabricante, mas também pela forma como organizam o trabalho e o acesso aos recursos da máquina.

> [!warning] Atenção
> **Monousuário**, **multitarefas** e **multiusuário** não significam a mesma coisa:
> - monousuário = quantidade de usuários;
> - multitarefas = quantidade de tarefas em execução;
> - multiusuário = vários usuários acessando o sistema.

> [!tip] Resumindo
> Os tipos de sistema operacional mostram como o sistema administra usuários e tarefas.

## 6. Ambiente operacional e interface gráfica

> [!info] Conceito
> O **ambiente operacional** integra o sistema operacional ao usuário, principalmente por meio da **interface gráfica**.

A interface gráfica, também chamada de GUI, facilita o uso do computador porque evita que o usuário precise memorizar comandos complexos. Em vez disso, ele utiliza elementos visuais para interagir com o sistema. O material destaca quatro componentes principais dessa interface: **ícones**, **cursor**, **menus** e **janelas**.

Esses elementos tornam o uso mais intuitivo. Os ícones representam arquivos, programas e comandos; o cursor indica a posição controlada pelo mouse; os menus apresentam opções; e as janelas delimitam o espaço das aplicações. Isso simplifica a execução de tarefas e melhora a experiência de uso.

> [!tip] Resumindo
> A interface gráfica torna o computador mais fácil de usar ao substituir comandos complexos por elementos visuais.

---

## 7. Recursos facilitados pela interface gráfica

> [!info] Conceito
> A interface gráfica não serve apenas para “enfeitar” o sistema: ela facilita ações práticas no uso diário.

O material cita algumas operações favorecidas pela interface gráfica. O **switching** permite alternar de um programa para outro sem encerrar a aplicação atual. O **windowing** possibilita sobrepor e visualizar diferentes janelas ao mesmo tempo. O **cut-and-paste** corresponde a mover ou copiar trechos entre janelas. Já a **concorrência** indica que duas ou mais aplicações podem se revezar no uso do processador.

Esses recursos mostram como a interface gráfica melhora a produtividade e torna o uso do computador mais acessível, especialmente para quem não domina comandos textuais complexos.

> [!tip] Resumindo
> A interface gráfica facilita a troca entre programas, o uso de várias janelas e a realização de tarefas simultâneas.

## 8. Tradutores de linguagem

> [!info] Conceito
> **Tradutores** são softwares básicos que convertem linguagens de programação em linguagem de máquina, que é a linguagem entendida pelo computador.

O capítulo apresenta três tipos de tradutores. O **compilador** lê e analisa o código-fonte e gera um programa em linguagem de máquina. O **interpretador** traduz e executa os comandos linha por linha, sem gerar uma versão executável final. O **montador** transforma linguagem de baixo nível, como assembly, em linguagem de máquina.

Em termos simples, esses tradutores são necessários porque as pessoas programam em linguagens compreensíveis para humanos, mas o computador executa instruções em nível muito mais básico.

> [!warning] Atenção
> É fácil confundir os tradutores:
> - **compilador** = traduz o programa e gera resultado;
> - **interpretador** = traduz e executa aos poucos;
> - **montador** = trabalha com linguagem de baixo nível.

> [!tip] Resumindo
> Os tradutores fazem a ponte entre a linguagem escrita por programadores e a linguagem que o computador realmente entende.

## 9. Software aplicativo

> [!info] Conceito
> **Software aplicativo** é o programa criado para ajudar o usuário a realizar tarefas específicas.

O material explica que essa é a principal característica do software aplicativo: ele existe para atender diretamente às necessidades do usuário. Exemplos citados incluem processadores de texto, planilhas de cálculo e bases de dados. Em outras palavras, são programas usados para produzir documentos, organizar números, registrar informações e automatizar atividades.

Também é destacado que os aplicativos para computadores de mesa e notebooks costumam ser chamados de aplicativos para escritório, enquanto os desenvolvidos para dispositivos móveis são chamados de aplicativos móveis.

> [!tip] Resumindo
> O software aplicativo não mantém o computador funcionando por si só; ele usa a base do sistema para ajudar o usuário a trabalhar.

## 10. Formas de aquisição e tipos de licença de software aplicativo

> [!info] Conceito
> O material apresenta diferentes formas de disponibilização e uso de softwares aplicativos.

O **freeware** é distribuído gratuitamente, mas continua protegido por direitos autorais, e o desenvolvedor pode impor restrições. O **software de domínio público** não tem proteção por direitos autorais e pode ser usado e modificado sem restrições. O **software livre** permite acesso ao código-fonte, o que possibilita estudar, corrigir erros e fazer melhorias. O **shareware** é distribuído gratuitamente de forma parcial ou para teste, mas o usuário deve pagar para obter autorização completa ou documentação. Já o **pacote de softwares** é protegido por direitos autorais e possui custo mais alto que o shareware. Por fim, o **software pirata** é uma cópia ilegal de software comercial.

Essa parte do conteúdo é importante porque mostra que o termo “gratuito” não significa sempre a mesma coisa. A forma de uso, de modificação e de distribuição muda conforme o tipo de licença.

> [!warning] Atenção
> O material mostra diferenças que costumam gerar confusão:
> - **freeware** = gratuito, mas com direitos autorais;
> - **software livre** = permite acesso ao código-fonte;
> - **domínio público** = sem restrições de direitos autorais;
> - **shareware** = teste gratuito, com pagamento para uso completo;
> - **software pirata** = cópia ilegal.

> [!tip] Resumindo
> Nem todo software gratuito oferece as mesmas liberdades. O tipo de licença define o que o usuário pode ou não pode fazer.

## 11. Plataformas de sistemas operacionais

> [!info] Conceito
> O texto apresenta três plataformas de sistemas operacionais de destaque: **Windows**, **MAC OS** e **Linux**.

O material informa que a Microsoft desenvolve os sistemas operacionais da plataforma Windows, conhecida pela interface gráfica e pela ampla utilização em computadores. A Apple desenvolve o MAC OS, que é restrito aos computadores da própria marca e se destaca pela qualidade gráfica. Já o Linux é apresentado como uma plataforma baseada em software livre, com várias comunidades de desenvolvimento e bom gerenciamento de memória, permitindo uso satisfatório até em hardwares menos avançados.

O objetivo dessa comparação é mostrar que os sistemas operacionais também podem ser diferenciados por plataforma, fabricante e características gerais de uso.

> [!tip] Resumindo
> Windows, MAC OS e Linux são exemplos de plataformas de sistemas operacionais com características e contextos de uso distintos.

## 12. Categorias de softwares aplicativos

> [!info] Conceito
> Os softwares aplicativos podem ser classificados de acordo com sua finalidade de uso.

O material apresenta várias categorias. Os **aplicativos de escritório** incluem processadores de texto, planilhas, utilitários, ferramentas de comunicação e gerenciadores de informações pessoais. Os **aplicativos administrativos** envolvem faturamento, folha de pagamento, estoque, produção e contabilidade. Os **aplicativos de automação comercial** incluem reservas, contas correntes, pontos de venda e caixas automáticos. Os **aplicativos técnico-científicos** são voltados para cálculos de estruturas, controle de projetos e problemas de engenharia. Os **aplicativos de automação industrial** servem para programar e controlar máquinas e processos. O material também inclui a categoria de **entretenimento**, como jogos e música.

Essa classificação mostra que os aplicativos existem para atender contextos muito diferentes, desde tarefas comuns de escritório até atividades industriais e científicas.

> [!tip] Resumindo
> Os aplicativos são organizados conforme o problema que resolvem: escritório, gestão, comércio, técnica, indústria ou entretenimento.

## 13. Exemplos de softwares aplicativos mais comuns

> [!info] Conceito
> Alguns softwares são considerados padrão no mercado por serem amplamente usados em atividades cotidianas e profissionais.

Entre os exemplos citados, os **processadores de texto** permitem criar, editar, armazenar e imprimir documentos com textos e gráficos. O material cita como exemplos o **Microsoft Word** e o **Write** do BR-Office. As **planilhas eletrônicas** organizam dados em linhas e colunas, permitem cálculos automáticos e ajudam na análise de informações; os exemplos apresentados são **Excel** e **Calc**. Os **softwares de apresentação** auxiliam na construção de materiais visuais com gráficos, mapas e tabelas, sendo citados **PowerPoint** e **Impress**. Também são mencionados os **navegadores de internet**, como Google Chrome, Mozilla Firefox, Internet Explorer e Opera.

Esses exemplos reforçam a ideia de que os aplicativos são ferramentas de produtividade e apoio à tomada de decisão.

> [!tip] Resumindo
> Processadores de texto, planilhas, programas de apresentação e navegadores são exemplos de aplicativos amplamente usados no dia a dia.

## 14. A relação entre hardware, software e usuário

> [!info] Conceito
> O material usa a ideia de uma **pirâmide** para explicar a relação entre hardware, software e usuário.

Na base da pirâmide estão os recursos de **hardware**, ou seja, os componentes físicos do computador. No meio está o **software**, que realiza a interação entre a máquina e o usuário. No topo está o **usuário**, que possui demandas e busca resolvê-las por meio dos programas disponíveis. O exemplo apresentado é o de João usando o Microsoft Word para digitar um documento.

Essa representação ajuda a compreender que o software ocupa uma posição intermediária fundamental: ele conecta a estrutura física da máquina às necessidades práticas das pessoas.

> [!tip] Resumindo
> O usuário depende do software para aproveitar o hardware e realizar suas tarefas.

## Síntese final

> [!summary] Síntese
> O material mostra que o **software** é a parte lógica responsável por fazer o computador executar tarefas. Ele se divide em **software básico**, que torna a máquina utilizável, e **software aplicativo**, que ajuda o usuário em atividades específicas. O **sistema operacional** é o principal exemplo de software básico, pois gerencia hardware, arquivos e a interação com o usuário. A **interface gráfica** facilita esse uso por meio de ícones, menus, janelas e cursor. Já os **aplicativos** atendem finalidades diversas, como escrever textos, calcular, apresentar informações, navegar na internet, administrar empresas, controlar processos técnicos ou até oferecer entretenimento. O conteúdo também diferencia formas de aquisição e licenciamento, como **freeware**, **software livre**, **domínio público**, **shareware** e **software pirata**, destacando que nem todo software gratuito oferece as mesmas permissões de uso e modificação.
